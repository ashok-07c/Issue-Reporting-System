# Local Complaint / Issue Reporting System

A modern frontend-based civic issue reporting platform where citizens can report local problems and track complaint status, while administrators can manage and resolve issues through a dedicated admin dashboard.

---

## 🚀 Features

### 👤 User Features

* User Registration & Login
* Report Local Complaints
* Track Complaint Status
* View Complaint History
* Responsive User Dashboard
* Dynamic Complaint Statistics

### 🛠️ Admin Features

* Admin Login
* Complaint Management Dashboard
* View All Complaints
* Update Complaint Status
* Analytics Overview
* Responsive Admin Panel

---

## 🧰 Technologies Used

* HTML5
* CSS3
* JavaScript
* LocalStorage (Frontend Database)
* Responsive Design
* iframe-based Dashboard Navigation

## 📌 How the System Works

1. Users register and login.
2. Users submit complaints through the report page.
3. Complaints are stored using LocalStorage.
4. Admin reviews complaints in the admin dashboard.
5. Admin updates complaint status.
6. Users track progress through the tracking page.
